# CodeAlpha Python Programming Internship Tasks

This repository contains the Python programming tasks completed as part of the CodeAlpha Internship.

## Tasks Completed

### 1. Hangman Game
- A text-based Hangman game.
- Randomly selects a word from a predefined list.
- Allows up to 6 incorrect guesses.

### Concepts Used
- Python
- Random Module
- Loops
- If-Else Statements
- Lists
- Strings


### 2. Stock Portfolio Tracker
- Calculates the total investment value of selected stocks.
- Uses a predefined dictionary of stock prices.
- Saves the total investment to a text file.

### Concepts Used
- Dictionary
- User Input
- File Handling
- Arithmetic Operations


### 3. Basic Chatbot
- A simple rule-based chatbot.
- Responds to greetings and common questions.
- Ends the conversation when the user types **bye**.

### Concepts Used
- Functions
- Loops
- If-Elif Statements
- User Input


## Technologies Used

- Python 3
- Visual Studio Code
